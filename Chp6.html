Chapter 6: Programming Languages
Machine language Understand the problem
debugging the process of finding and fixing errors or bugs in the source code of any software
assembler a type of computer program that takes in basic instructions and converts them into a pattern of bits that the computer's processor can use to perform basic operations
assembly language a type of low-level programming language that is intended to communicate directly with a computer's hardware
translator a program that converts source code into object code
compiler computer software that translates (compiles) source code written in a high-level language (e.g., C++) into a set of machine-language instructions that can be understood by a digital computer's CPU
interpreter a computer program that directly executes instructions written in a programming or scripting language, without requiring them previously to have been compiled into a machine language program
cross-platform software a type of application / program / software that works on various operating systems or devices, which are often called platforms
natural language a machine learning technology that gives computers the ability to interpret, manipulate, and comprehend human language.
formal language a set of strings of symbols drawn from a finite alphabet
machine independence A term applied to software that is not dependent on the properties of a particular machine, and can therefore be used on any machine
high-level programming language a programming language such as C, FORTRAN, or Pascal that enables a programmer to write programs that are more or less independent of a particular type of computer
object-oriented programming (OOP) a style of programming characterized by the identification of classes of objects closely linked with the methods (functions) with which they are associated
Scripting language a system of notation for writing computer programs
primitive data type a set of basic data types from which all other data types are constructed
data structure a specific way of organizing data in a specialized format on a computer so that the information can be organized, processed, stored, and retrieved quickly and effectively
function a sequence of program instructions that performs a specific task, packaged as a unit
local variable a variable that is given local scope
global variable variables that are accessible globally, or everywhere throughout the program
source program programming statements that are created by a programmer with a text editor or a visual programming tool and then saved in a file
object program an abstract data type with the addition of polymorphism and inheritance
Data types
integer a data type that represents some range of mathematical integers
float a function or reusable code in the Python programming language that converts values into floating point numbers
string is traditionally a sequence of characters, either as a literal constant or as some kind of variable
Pyhon collection data structures
list Lists are used to store multiple items in a single variable.
tuple Tuples are used to store multiple items in a single variable.
set Sets are used to store multiple items in a single variable.
dictionary Dictionaries are used to store data values in key:value pairs
